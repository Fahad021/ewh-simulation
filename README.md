# ewh-simulation
Simulation of a network of electric water heaters

## Setup

### Mac without Virtualenv

```bash
$ brew install python3
$ pip3 install -r requirements.txt
```

### Mac with Virtualenv

```bash
$ brew3 install python
$ pip3 install virtualenv
$ virtualenv venv
$ source venv/bin/activate
$ pip3 install -r requirements.txt
```
